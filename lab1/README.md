# Lab 1

## Overview
This lab contains two simple C programs demonstrating basic input/output and arithmetic operations.

## Directory Structure
- `Program1/`
  - `lab1.c` — reads two integers and prints their sum.
  - `lab1.i` — preprocessed output for `lab1.c`.
  - `lab1.s` — assembly output for `lab1.c`.
- `Program2/`
  - `lab1_PI.c` — calculates the area of a circle using a constant value for pi.
  - `lab1_PI.i` — preprocessed output for `lab1_PI.c`.
  - `lab1_PI.s` — assembly output for `lab1_PI.c`.

## Program Descriptions

### Program 1 — Sum of Two Numbers
- Reads two integers from the user.
- Computes the sum.
- Prints the result.

### Program 2 — Area of a Circle
- Reads the radius of a circle from the user.
- Uses the macro constant `PI`.
- Computes and prints the area with two decimal places.

## Build and Run

### Compile Program 1
```bash
cd "lab1/Program1"
gcc lab1.c -o lab1
./lab1
```

### Compile Program 2
```bash
cd "lab1/Program2"
gcc lab1_PI.c -o lab1_PI
./lab1_PI
```

## Notes
- The `.i` files are generated by the C preprocessor.
- The `.s` files are generated by the compiler as the assembly listing.
- Use `gcc -E` to generate `.i` files and `gcc -S` to generate `.s` files if needed.
